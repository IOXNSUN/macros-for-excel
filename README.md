# 📝 4️⃣ macros-for-excel


## О проекте
Макросы для автоматизации рутинных задач в Excel.  
Ускоряют обработку данных и интеграцию с внешними системами.

## Технологии
- VBA, Excel
- Python + COM (опционально)
- CSV / XLSX


## Установка и запуск
- Открыть Excel
- Включить макросы
- Использовать кнопки макросов
